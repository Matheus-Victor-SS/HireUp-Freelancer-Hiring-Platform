# HIREUP-Site-de-contrata-o-de-freelancers
HireUp é uma plataforma digital desenvolvida em WordPress com Elementor e funcionalidades personalizadas em PHP, criada para facilitar e tornar mais segura a contratação de freelancers. O projeto integra frontend moderno, backend customizado e banco de dados, sendo acompanhado por um documento de pesquisa com detalhes e prints
